# Getting Started with OOPress

Welcome to OOPress! This guide will help you get started with your new CMS.

## First Steps

After installation, you can log in to your admin panel at:

https://your-domain.com/login


Use the admin credentials you created during installation.

## Dashboard Overview

The admin dashboard gives you an overview of your site:

- **Posts** - Total, published, and draft posts
- **Users** - Registered users on your site
- **Recent Posts** - Latest content created
- **Recent Users** - Newest registered users

## Creating Your First Post

1. Click **Posts** in the left sidebar
2. Click **Add New Post**
3. Enter your post title
4. Write your content using the editor
5. Choose a format (TinyMCE, HTML, Markdown, or PHP)
6. Select categories and tags
7. Set SEO meta data (optional)
8. Click **Publish**

## Managing Pages

Pages are for static content like "About Us" or "Contact":

1. Click **Pages** in the left sidebar
2. Click **Add New Page**
3. Enter the page title and content
4. Set parent page (for hierarchical pages)
5. Choose a template (if available)
6. Enable "Show in Menu" to display in navigation
7. Set menu order (lower numbers appear first)
8. Publish

## Uploading Media

1. Click **Media** in the left sidebar
2. Click **Upload File**
3. Select an image or file
4. The file will appear in your media library
5. Click **Copy URL** to get the file URL for use in posts

## Managing Categories and Tags

Organize your content with categories and tags:

- **Categories** - Broad topic grouping (hierarchical)
- **Tags** - Specific keywords (non-hierarchical)

To add a category:
1. Click **Categories**
2. Click **Add New Category**
3. Enter name and description
4. Save

## User Management

As an admin, you can manage all users:

1. Click **Users** in the left sidebar
2. View all registered users
3. Edit user roles and status
4. Change passwords if needed

User roles:
- **Admin** - Full access to everything
- **Editor** - Can create and edit all posts
- **Author** - Can create and edit own posts
- **Subscriber** - Can only read content

## Site Settings

Configure your site in **Settings**:

- **General** - Site title, tagline, timezone, date format
- **Reading** - Posts per page, excerpt settings
- **Comments** - Enable comments, moderation settings
- **SEO** - Meta description, keywords
- **Media** - Upload limits, allowed file types
- **Advanced** - Maintenance mode
